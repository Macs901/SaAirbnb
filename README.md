# Gestao Porto Rio - Projeto Airbnb

Projeto de gestao profissional de aluguel de temporada no Rio de Janeiro.

## Estrutura do Projeto

```
SaAirbnb/
├── docs/                           # Documentacao
│   ├── planejamento/               # Planejamento estrategico
│   │   ├── PRD_Gestao_Airbnb.md    # Product Requirements Document
│   │   └── Checklist_Lancamento.md # Checklist operacional 30 dias
│   ├── comercial/                  # Materiais comerciais
│   │   └── Proposta_Comercial.md   # Proposta para proprietarios
│   └── mercado/                    # Analises de mercado
│       └── Validacao_Mercado_2026.md # Dados validados 2025/2026
├── site/                           # Landing page
│   ├── index.html                  # Pagina principal (atual)
│   ├── assets/                     # Recursos estaticos
│   │   ├── css/                    # Estilos
│   │   ├── js/                     # Scripts
│   │   └── images/                 # Imagens
│   └── _archive/                   # Versoes antigas
│       └── Landing_Page_v1.html    # Versao anterior
├── propostas/                      # Propostas para clientes
│   └── [propostas individuais]     # PDFs por cliente
└── README.md                       # Este arquivo
```
